# FocusWave — Binaural Beats

Chrome extension (Manifest V3) that generates binaural beats for focus, meditation, relaxation, and sleep.

## Install (dev)

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked** and select the `focuswave/` folder

## Structure

- `focuswave/manifest.json` — MV3 manifest
- `focuswave/background/` — service worker
- `focuswave/offscreen/` — offscreen audio document
- `focuswave/popup/` — popup UI
- `focuswave/welcome/` — onboarding page
- `focuswave/icons/` — extension icons
